# ModelEditor
[![Release](https://img.shields.io/github/v/release/Xqedii/ModelEditor)](https://github.com/Xqedii/ModelEditor/releases/latest) 
[![Discord](https://img.shields.io/badge/Discord-Join-7289DA?logo=discord&logoColor=white)](https://discord.com/invite/6JQfeQEB4W)

**ModelEditor is a Minecraft Item Display and Block Display editor written in Skript**

This code allows you to edit Item Display and Block Display easily in-game without the need to type long commands. The editor enables model rotation, movement, and scaling, so you can arrange them however you like!

> [!NOTE]
> This project is not finished and is not recommended for use on production servers
> 
-----
